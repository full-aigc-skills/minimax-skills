## 📖 简介

**minimax-skills** 是一组 AI 编码智能体技能，属于 [Full AIGC Skills](https://github.com/full-aigc-skills) 生态。包含 **3 个技能**。

## 📦 安装

```bash
npx skills add full-aigc-skills/minimax-skills
```
